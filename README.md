# 项目简介

本系统是基于BERT与c-TF-IDF算法的深度学习语义主题建模，旨在提供文本数据的深入分析和主题抽取。主要功能模块包括文本预处理、BERT特征提取、c-TF-IDF权重计算及主题模型构建。技术特点在于融合了BERT的强大语义理解能力和c-TF-IDF在主题词权重分配上的优势，提升了主题抽取的准确性和效率。系统附带了完整的源码、文档说明及所需全部资料，方便用户理解和使用。


## 核心功能
- 文本预处理模块
- BERT特征提取模块
- c-TF-IDF权重计算模块
- 主题模型构建模块

## 适用场景
- 文本数据的深入分析
- 主题抽取任务
- 学术研究
- 文档分类与聚类

## 技术特点
- 融合BERT的强大语义理解能力
- 结合c-TF-IDF在主题词权重分配上的优势
- 提升主题抽取的准确性和效率
- 提供完整的源码和文档说明
- 包含系统所需的全部参考资料
